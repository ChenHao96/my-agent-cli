# my-agent-cli

## 项目简介

`my-agent-cli` 是一个基于 Python 开发的 AI Agent 命令行工具，集成了多种系统工具和文件操作功能。该项目通过 OpenAI API 与 Qwen/qwen3.5-9b 大模型进行交互，实现智能化的对话和任务执行能力。

## 主要特性

### 🚀 核心功能
- **智能对话**: 基于 OpenAI API 与大语言模型（Qwen/qwen3.5-9b）进行实时交互
- **工具调用**: 支持多种系统工具自动调用和执行
- **流式处理**: 完整的上下文管理和消息历史记录维护

### 🛠️ 内置工具

#### 系统信息工具
| 工具名称 | 功能描述 |
|---------|---------|
| `get_date` | 获取当前日期 |
| `get_ip` | 获取当前网络 IP |
| `get_ip_info` | 获取 IP 的地理位置信息 |

#### 目录操作工具
| 工具名称 | 功能描述 |
|---------|---------|
| `dir_location` | 获取当前目录位置 |
| `dir_ls` | 列出当前目录下所有文件（包括文件夹） |
| `dir_create` | 创建新文件夹 |

#### 文件操作工具
| 工具名称 | 功能描述 |
|---------|---------|
| `file_create` | 创建空文件 |
| `file_read` | 读取文件内容 |
| `file_write` | 写入/覆盖文件内容 |
| `file_append` | 向文件追加内容 |
| `file_read_lines` | 读取文件的指定行内容 |

#### 系统命令工具
| 工具名称 | 功能描述 |
|---------|---------|
| `os_bash` | 执行命令行命令（最低优先级，需用户确认） |

## 安装与运行

### 前置要求
- Python 3.x
- OpenAI API Key

### 环境准备

1. **创建虚拟环境**（如果尚未创建）:
```bash
python -m venv .venv
```

2. **激活虚拟环境**:
```bash
# Windows PowerShell
.\.venv\Scripts\Activate.ps1

# Windows CMD
.venv\Scripts\activate
```

3. **安装依赖**:
```bash
pip install -r requirements.txt
```

### 运行项目

直接运行主程序：
```bash
python main.py
```

程序启动后会进入交互模式，提示符为 `>>`，可以输入各种指令与大模型进行对话。

## 配置说明

### 系统提示词 (`.agent/system_prompt.md`)

包含以下核心要求：

#### 📝 通用要求
- **思考语言**: 必须使用英语进行思考和推理
- **回复语言**: 回复内容必须使用中文，专有名词保留原始形式（例如："Token"不应翻译为令牌/词元）
- **决策树原则**: 用户问题简单时逐步确认，明确步骤后再执行
- **os_bash 优先级最低**: 优先使用专用工具，仅在无对应工具时使用系统命令
- **安全确认**: 所有删除文件、命令行操作需用户明确确认

#### 📂 文件操作流程（Copy File / Copy Folder / Move File）
- 目标不存在：创建 → 写入内容 → 完成
- 目标已存在：询问用户是否覆盖 → 确认后执行或取消
- 文件夹包含子文件夹时递归复制/移动

### 用户提示词 (`.agent/user_prompt.md`)

#### 📋 文档要求
- **默认使用 Markdown**: 除非用户指定其他文档类型
- **图表绘制**: 不允许多文本构建，应使用 Mermaid 格式
- **问答记录保存**: 用户问题和回答需保存在 `user_QA.md` 文件中（当前目录）

#### 📖 user_QA.md 附加格式
```markdown
## User Q&A Records

> **Question**: 今日是什么日期？  
> **Answer**: 今天是 2026-05-18  

> **Question**: 今日星期几？  
> **Answer**: 今日是星期一  

> ...

> **Question**: 最近的周六在哪天？  
> **Answer**: 2026-05-16(周六)最近, 下一个周六在 2026-05-23  
```

## API 配置

在 `main.py` 中配置了以下 OpenAI 客户端参数：

```python
client = OpenAI(
    api_key="llm-studio",
    base_url="http://localhost/v1",
)
```

如需更改模型或其他设置，请编辑 `main.py` 中的以下变量：

- `api_key`: API 密钥
- `base_url`: API 服务地址
- `model`: 使用的模型（默认：`qwen/qwen3.5-9b`）
- `reasoning_effort`: 推理难度（默认：`off`）
